## Arsitektur Aplikasi

DoaKu Kids menggunakan arsitektur monolitik terintegrasi (_Laravel Monolith_) di mana bagian frontend dibangun menggunakan Laravel Blade dan Tailwind CSS di dalam satu proyek Laravel yang sama dengan backend.

**Aplikasi Terintegrasi — DoaKu Kids**

- Deskripsi Singkat: Aplikasi web interaktif untuk anak-anak yang menyajikan daftar doa harian dengan visual yang menarik serta mengelola database doa, kategori, dan integrasi dengan API pihak ketiga di sisi backend.
